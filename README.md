## 1. 프로젝트 개요
<b> 주제 : 일대다 수업 방식 중 발생하는 학생과 교사간 소통 문제 해결 </b>
- 학생들의 풀이완료/질문/도움 상태를 버튼을 통해 입력받아 
  각각 초록/노랑/빨강 Led를 하드웨어로 출력하는 서비스 제공
- 하드웨어 뿐만 아니라 웹 서비스를 통해 교사 컴퓨터로 실시간 점등 상태 확인 가능
- 문제별 Led 점등 상태를 저장하여 풀이 완료 시간과 질문 수를 통계치로 확인 가능
